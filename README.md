
# Smart Factory – Power BI Dashboard

Ce projet analyse les données d'une usine intelligente pour :
- Surveiller les machines
- Détecter les anomalies
- Analyser la maintenance
- Evaluer la performance énergétique

## Contenu
- SmartFactory.pbix : Dashboard Power BI
- Dataset : factory_sensor_200.csv
- Theme : SmartFactoryTheme.json
- Notebook Python : nettoyage des données
- Captures d’écran
